# Dongsheng (David) Han — Personal Website

Personal research website for Dongsheng (David) Han, Ph.D., a Stanford Bioengineering postdoctoral scholar and NIH/NIA F99/K00 Fellow.

## Website

This is a static website designed for GitHub Pages. The repository root contains:

- `index.html` — the complete website
- `Dongsheng_Han_CV.pdf` — downloadable two-page CV
- `.nojekyll` — ensures GitHub Pages serves the static files directly

## Publish with GitHub Pages

1. Push this repository to GitHub.
2. Open the repository's **Settings → Pages**.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Select the `main` branch and the `/(root)` folder, then save.

For a personal root URL, name the repository `<github-username>.github.io`.
